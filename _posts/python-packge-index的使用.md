python packge index（PyPI）是Python编程语言的软件存储库。可用下载各种常用库，以及自己制作并上传库

## 官方网站

https://pypi.python.org

## 安装

pip3 install pymysql

安装完的库在python根目录下的lib\sitepackgs\目录下

如果需要在命令行窗口直接运行pip3 需要配置python根目录下的script目录为环境变量
